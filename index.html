Styleguide

Titles

This is a styleguide, it displays all the styles for a site in one place allowing for easy reference, perusal and updating. We'll begin with headers (at the top of this page is an <h1> - this should only be used for the site title)

Second Level Header

Second level headers, <h2> elements, are page titles, article titles - main content titles essentially. More than one can be used per page.

Third Level Header

Third level headers, <h3> elements, should be used as a subheader to an <h2> or as a header at a lower level to an <h2>.

Fourth Level Header

Third level headers, <h4> elements, should be used for any header at a lower level to an <h3>.

Fifth Level Header

Third level headers, <h5> elements, should be used for any header at a lower level to an <h4>.

Sixth Level Header

Third level headers, <h6> elements, should be used for any header at a lower level to an <h5>.

Paragraphs

Paragraphs contain text content, wrapped in <p> tags, if they contain quotes they can be wrapped in a <blockquote>. These can also be associated with a <cite> element to demarkate the source of the quote if necessary.

Lists

Lists come in 3 flavours. Unordered lists and ordered lists, <ul> and <ol> tags respectively, are lists of individual items and are self explanatory in their uses. Definition Lists, <dl> tags, are lists of pairs of items with a parent/child relationship. Traditionally used for dictionary style definitions they can be applied to other scenarios too.

Unordered List

  • List item
  • Another list item
  • Another list item, with sub items (unordered)
    • Sub list item
    • Another sub list item
  • List item with sub items (ordered)
    1. Sub item A
    2. Sub item B

Ordered List

  1. List item
  2. Another list item
  3. Another list item, with sub items (unordered)
    • Sub list item
    • Another sub list item
  4. List item with sub items (ordered)
    1. Sub item A
    2. Sub item B

Inline Content

Certain tags may be used anywhere within other elements, these are inline elements. These include abbr, acronym, dfn, time code, kbd, var, cite, del, ins, sub, sup, em, i, strong, b, small, mark

Anchor Tags (Links)

An element of note (yeah it gets its own section), the <a> tag is used to provide links both internal and external. In HTML versions prior to HTML5 it could only be wrapped around inline elements and content but as of HTML5 we can apply it around block level elements too, and multiple ones at that.
